# Contributing

CodeX Client is an archived project.

That means this repository is public so the source stays available, but it is not under active official development anymore.

If you want to build on it:

- feel free to fork it
- feel free to learn from it
- feel free to continue the idea in your own direction

A few expectations:

- keep your changes clear and readable
- do not reintroduce old multi-version scaffolding unless you genuinely plan to maintain it
- if you publish a fork, maintain it under your own name, support, and release responsibility
- if you make major gameplay or feature changes, document them clearly in your own README

For this archived repo itself, small cleanup and maintenance-oriented improvements are the most appropriate kind of contribution.
